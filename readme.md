# このプロジェクト
勉強用として、機械学習アプリを作りました。バックエンドはGoogleCloud上にフロントエンドはFirebaseでホスティングする予定です 音楽雑誌、音楽書籍から文章を抜き出し、前処理後、WordEmbeddingを実装したライブラリで学習し、アーティストの近傍探索ができるアプリとなります。 このレポジトリはバックエンドのプロジェクトとなります。
